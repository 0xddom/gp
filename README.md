Genetic programming demos
=================

GP experiments in classic Koza and OOGP

## Setup

1. Install `gem install gp`.

## Usage

    gp help

## Development

When hacking on this gem, the REPL `pry` comes in handy. You can load the
contents of the gem with `pry --gem`.

To test the CLI, run

    ruby -Ilib bin/gp
